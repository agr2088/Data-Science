# Multiple Linear Regression Assignments

This folder contains instructor-evaluated assignments focused on
building and evaluating a Multiple Linear Regression (MLR) model
to understand relationships between multiple independent variables
and a continuous target variable.

## Files Included
- **Assignment_7.ipynb**  
  Python implementation including:
  - Understanding the dataset and feature selection
  - Data preprocessing and cleaning
  - Splitting data into training and testing sets
  - Building a Multiple Linear Regression model
  - Evaluating model performance using appropriate metrics
  - Interpreting regression coefficients
  - Drawing conclusions based on model results

## Skills Demonstrated
- Multiple Linear Regression (MLR)
- Feature selection and preprocessing
- Model training and evaluation
- Interpretation of regression coefficients
- Applied machine learning using scikit-learn
