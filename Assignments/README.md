# Data Science Assignments

This folder contains **instructor-evaluated assignments** completed as part of
my Data Science coursework, focusing on building strong fundamentals and
practical problem-solving skills.

## Contents
- **python_basics/**  
  Core Python programming concepts including control flow, functions,
  loops, and basic real-world problem solving.

- **numpy_pandas/**  
  Assignments related to Python data structures, NumPy array operations,
  and basic data manipulation using Pandas.

- **data_visualization/**  
  Visual exploration of data using Matplotlib and Seaborn to derive insights.

- **statistics/**  
  Statistical concepts including descriptive statistics, correlations,
  and hypothesis testing.

- **machine_learning/**  
  Supervised and unsupervised machine learning assignments covering model
  training, evaluation, and interpretation.

- **deep_learning_intro/**  
  Introductory assignments covering basic neural network concepts.

## Purpose
These assignments demonstrate my understanding of data science concepts,
ability to translate theory into code, and experience working with
industry-standard tools and libraries.
