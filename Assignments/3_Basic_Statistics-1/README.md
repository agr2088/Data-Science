# Basic Statistics Assignments

This folder contains instructor-evaluated assignments focused on
descriptive statistics, data visualization, and basic data preprocessing
using a sales and discounts dataset.

## Files Included
- **Basic statistics.docx**  
  Covers problem statements related to descriptive analytics,
  statistical summaries, and visualization requirements.

- **Assignment_3.ipynb**  
  Python implementations including:
  - Identification of numerical and categorical columns
  - Calculation of mean, median, mode, and standard deviation
  - Interpretation of statistical measures
  - Histogram analysis for numerical data
  - Box plot analysis to detect outliers
  - Bar chart analysis for categorical variables
  - Summary of key insights from data analysis

## Skills Demonstrated
- Descriptive statistical analysis
- Data visualization techniques
- Interpretation of data distributions
- Outlier detection using box plots
- Exploratory data understanding
