# Basic Statistics & Data Analysis Assignment

This folder contains an instructor-evaluated assignment focused on
applying statistical concepts and data manipulation techniques to a
real-world hospital patient and billing dataset.

## Files Included
- **Basic statistics-2.docx**  
  Contains the problem statement and tasks related to hospital patient
  data analysis and billing operations.

- **Assignment_4.ipynb**  
  Python implementation including:
  - Loading and inspecting datasets using Pandas
  - Selecting and dropping relevant columns
  - Handling missing values using statistical measures
  - Removing duplicate records
  - Grouping data to analyze department-wise billing
  - Merging patient and billing datasets
  - Row-wise and column-wise data concatenation

- **Patient_Data.csv**  
  Dataset containing patient admission and medical details.

- **Billing_Data.csv**  
  Dataset containing billing-related information.

## Skills Demonstrated
- Data cleaning and preprocessing
- Handling missing and duplicate data
- GroupBy operations and aggregation
- Dataset merging and concatenation
- Applied statistics for real-world data
- Pandas-based data analysis
