# Data Transformation Assignments

This folder contains instructor-evaluated assignments focused on
transforming raw data into a suitable format for machine learning
and statistical modeling.

## Files Included
- **Assignment_9.ipynb**  
  Python implementation including:
  - Identifying numerical and categorical features
  - Handling missing values
  - Encoding categorical variables
  - Feature scaling and normalization
  - Preparing transformed datasets for modeling
  - Verifying data consistency after transformation

## Skills Demonstrated
- Data transformation techniques
- Feature encoding
- Feature scaling and normalization
- Data preprocessing for machine learning
- Building clean and model-ready datasets
