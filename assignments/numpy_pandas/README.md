# Data Structures, Functions, NumPy & Pandas Assignments

This folder contains instructor-evaluated assignments focused on
Python data structures, functional programming, and foundational
data handling using NumPy and Pandas.

## Files Included
- **Data Structures, Functions, Numpy & Pandas.docx**  
  Contains assignment questions and conceptual tasks related to
  Python data structures and data manipulation.

- **Assignment_2.ipynb**  
  Python implementations including:
  - List, tuple, set, and dictionary operations
  - Conditional logic and control structures
  - Function-based problem solving
  - NumPy array creation and manipulation
  - Conversion of NumPy arrays to Pandas DataFrames
  - Basic data cleaning and exploration using Pandas

## Skills Demonstrated
- Python data structures
- Conditional statements
- Function implementation
- NumPy for numerical computing
- Pandas for data manipulation
- Data cleaning and exploration
