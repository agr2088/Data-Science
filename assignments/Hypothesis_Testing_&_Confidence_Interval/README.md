# Confidence Interval & Hypothesis Testing Assignments

This folder contains instructor-evaluated assignments focused on
inferential statistics, specifically confidence interval estimation
and hypothesis testing using real-world business and manufacturing
scenarios.

## Files Included
- **Confidence Interval.docx**  
  Covers problem statements related to estimation of population mean
  using confidence intervals under small sample conditions.

- **Assignment_6.1.ipynb**  
  Python implementation including:
  - Calculation of sample mean and standard deviation
  - Construction of confidence intervals using t-distribution
  - Confidence interval estimation using known population standard deviation
  - Interpretation of confidence interval results

- **Hypothesis testing.docx**  
  Contains problem statements related to testing business cost models
  using statistical hypothesis testing.

- **Assignment_6.2.ipynb**  
  Python implementation including:
  - Formulation of null and alternative hypotheses
  - Calculation of test statistics
  - Critical value and p-value comparison
  - Decision making based on significance level
  - Interpretation of statistical test results

## Skills Demonstrated
- Confidence interval estimation
- Use of t-distribution and z-distribution
- Hypothesis formulation and testing
- Statistical decision making
- Interpretation of inferential statistics
